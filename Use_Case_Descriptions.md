Actor – Description
회원 - 온라인 설문조사 플랫폼에 가입한 사용자. 설문 검색 및 응답, 응답 내용 조회·수정·취소를 수행한다.
비회원 - 플랫폼에 가입하지 않은 사용자. 회원가입을 통해 회원이 될 수 있다.
관리자 - 주어진 ID(admin)와 비밀번호(admin)로 로그인하는 시스템 관리자. 설문 등록·조회·삭제 및 응답 통계 조회를 수행한다.

Use Case - Description
회원가입 - 회원이 ID, 비밀번호, 이름, 전화번호, 이메일을 입력하여 시스템에 등록한다.
회원탈퇴 - 회원이 탈퇴를 요청하면 모든 이용 권한과 데이터가 삭제된다.
로그인 - 회원 또는 관리자가 ID와 비밀번호를 입력하여 시스템에 접속한다.
로그아웃 - 회원 또는 관리자가 시스템 접속을 종료한다.
설문 상세 조회 - 회원이 검색 결과에서 특정 설문을 선택하면 제목·설명·문항·응답항목·시작/마감 시각을 확인한다.
설문 응답 - 진행 중인 설문의 상세 정보 화면에서 응답 버튼이 활성화된 경우, 회원이 문항에 응답하고 제출한다.
응답 설문 조회 - 회원이 자신이 응답한 모든 설문의 제목·응답 내용·응답 일시를 한 화면에서 확인한다.
응답 수정 - 진행 중인 설문에 한해 회원이 기존 응답을 수정하고 최근 수정 날짜를 갱신한다.
응답 취소 - 진행 중인 설문에 한해 회원이 제출한 응답을 취소한다.
설문 등록 - 관리자가 설문 제목·설명·문항·응답항목·시작/마감 시각을 입력하여 설문을 등록한다.
설문 조회 - 관리자가 등록된 모든 설문 목록을 조회한다.
설문 상세 팝업 - 관리자가 설문 목록에서 특정 설문을 선택하면 상세 내용을 팝업 창으로 확인한다.
삭제 - 설문 리스트에서 특정 설문을 선택하여 삭제한다.
응답 통계 조회 - 관리자가 기간을 선택하면 해당 기간의 전체 설문 수·종료된 설문 수·총 응답 수를 조회한다.
